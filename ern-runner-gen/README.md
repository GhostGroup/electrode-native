# ern-runner-gen

This package is part of [Electrode Native][1].

The Electrode Native Runner Generator creates native projects to run a MiniApp.

## Usage

See the [documentation][2] and CLI help for more information:

```sh
ern run-android --help
ern run-ios --help
```
